# Bob-s-Talking-NPCs
Bob's Talking NPCs - A comprehensive standalone module for Foundry VTT V13 + D&amp;D 5e 5.2.4. Complete NPC interaction system featuring quests, dialogue trees with visual editor, merchants, factions, relationships, banking, hirelings, mounts, crime systems, and more. Replaces Simple Quest, Item Piles, and Monk's Active Tile Triggers.
